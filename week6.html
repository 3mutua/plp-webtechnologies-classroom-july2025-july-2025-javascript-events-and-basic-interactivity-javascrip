What are Functions?

Functions are like recipes - they contain instructions to do something.

Messages will appear here!

// This is how we create a function

  messageDisplay.textContent = "Hello, friend!";
}

// This is how we call (use) the function
sayHelloBtn.addEventListener('click', sayHello);

JavaScript FAQ

Click on the questions to learn more about JavaScript!

What is JavaScript? +

JavaScript is a programming language that makes websites interactive. It's what makes games, animations, and dynamic content possible on web pages!

Is JavaScript the same as Java? +

No, they are completely different programming languages! The name JavaScript was chosen because Java was popular when JavaScript was created.

What can I do with JavaScript? +

You can create games, web applications, interactive forms, animations, and so much more! Almost every modern website uses JavaScript.
